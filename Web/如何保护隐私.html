    # 如何保护隐私

    参考来源: 编程随想的博客

    # 防范社会工程学攻击

    案例: 徐玉玉电信诈骗案: 2016年8月, 徐玉玉高考结束被南京邮电大学英语专业录取. 在8月19号, 徐玉玉的母亲接到骗子的电话, 对方声称有2600元助学金, 并说这是发放助学金的最后一天, 但需要将其原有的9900元通过ATM机取出, 以此来激活银行卡, 再将所有的钱汇入一个指定账号, 到时候将助学金和原有的钱一起取出. 徐玉玉照做了, 骗子随后关机, 9900元学费被全部骗走. 当晚, 徐玉玉报警后在回来的路上, 心脏骤停, 最终抢救无效死亡.

    # 普及教育

    按照二八原理, 20%的简单培训就可以防范80%的潜在攻击. 随着时代的发展, 我们的工作和生活需要经常使用到计算机, 手机, 银行卡等等, 还有, 因为人是社会工程学攻击的主要对象, 所以对员工(大学生)进行基础的信息安全教育显得十分重要.

    # 严格的授权

    我们应该根据信息的重要程度, 定出若干级别的授权, 授权要尽量细化. 比如低级别的教务人员无法拿到全校学生的隐私信息.

    # 处理好信息垃圾

    对于涉及到重要信息的垃圾, 比如已经废弃的学生学号信息单, 要处理妥善

    # 行为审计

    上网行为审计
    主机行为审计

    # 个人软件和操作系统的防范

    案例:

    1. 2017年8月, Xshell被曝多版本被植入后门代码, 可能导致敏感信息泄露.
    2. 艳照门事件: 2008年1月, 多名香港女艺人不雅照片在互联网广泛流传. 香港警方调查后怀疑, 涉案照片原来保存在某明星的计算机上, 在送交电脑修理店时遭人非法复制照片, 更在互联网上被公开

    # 加密文件系统

    设置开机密码
    针对重要的文件或文件夹进行加密, 使得同一操作系统的其他用户无法读取文件内容

    # 使用虚拟机, 安卓模拟器

    有时候, 自己因为某些事情不得不使用那些可能会侵犯自己隐私的软件, 这个时候可以利用虚拟机的快照功能, 经常使用的虚拟机有VMWare Workstation, Virtual Box.

    # 避免使用高权限用户

    避免使用windows操作系统中的Administrators组的用户(Linux中的用户), 避免软件获得管理员权限.

    # 漏洞的防范

    案例: 永恒之蓝是美国国家安全局开发的漏洞利用程序, 于2017年4月被黑客组织泄露. 该工具利用445端口的文件分享协议的漏洞进行散播. 5月份WannaCry勒索软件利用这个漏洞大规模传播, 中国部分高校和政企专网计算机受到感染, 被迫停止工作.

    使用个人防火墙, 阻止电脑对外开放危险的端口
    定期升级操作系统的补丁
    定期升级更新软件
    计算机的摄像头其实很少会用到, 因此我们可以使用贴纸将其遮挡住

    # 密码安全

    案例: 2011年中国网站用户信息泄露事件: 2011年12月, CSDN遭黑客攻击, 600万用户信息(包括明文密码)泄露. 12月25日, 乌云漏洞平台曝出天涯社区4000万用户信息泄露(包括明文密码).

    # 非法获取密码的攻击手段

    木马盗取
    计算机遭到黑客入侵, 黑客在计算机安装了木马, 用户的一举一动都有可能被监视

    弱口令猜解
    用户在注册账号时, 使用了诸如123456, password的弱密码. 黑帽子可以通过字典爆破, 根据收集到的用户信息(比如生日, 手机号码)猜解用户的弱口令

    网络嗅探
    分析上网的数据, 获取上网过程中产生的隐私信息(比如公共wifi的嗅探)

    服务器端获取
    黑帽子通过攻击手段拿到服务器的权限, 而服务器居然使用明文存储用户的密码

    网络钓鱼
    结合社会工程学攻击, 诱骗到你的信息

    # 防范的方法

    使用密码管理工具
    不同的站点, 不同的账号不要使用一样的密码, 密码长度要大于8位, 不要使用纯数字或纯字母

    # 移动设备的隐私问题

    案例: 2019年, 美团等外卖平台, 被曝可能存在app麦克风窃听情况

    基于麦克风窃听
    泄露你的地理位置
    泄露你的生活规律

    尽量不要使用移动设备进行相关的隐私信息操作
