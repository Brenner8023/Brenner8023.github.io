    # XSS

    # 分类

    XSS全称为Cross Site Scripting. 中文名为跨站脚本攻击. 该漏洞发生在用户端, 是指在渲染过程中发生了不在预期过程中的javascript代码执行.

    # 同源策略

    同源策略限制了不同源之间进行资源交互, 是用于隔离潜在恶意文件的安全机制. 如果两个url的协议, 端口, 域名皆相同, 则认为它们同源.
