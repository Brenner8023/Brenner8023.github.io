    # babel

    配置文件.babelrc
    其中的presets字段设定转码规则

    {
	"presets": [
  	"es2015",
    "react"
  ],
  "plugins":[]
}

npm install --save-dev babel-presets-es2015

npm install --save-dev babel-presets-react

    babel-cli命令行转码

    npm install babel-cli --save-dev

babel example.js // 转码结果输出到标准输出

babel example.js --out-file compiled.js
babel example.js -o compiled.js

babel src --out-dir lib // src文件夹转码输出到lib目录
babel src -d lib

    # whistle

    https://github.com/avwo/whistle

    whistle基于node实现的跨平台web调试代理工具, 主要用于查看, 修改http, https, websocket的请求响应, 也可以作为http代理服务器使用.

    npm i -g whistle

whistle help
