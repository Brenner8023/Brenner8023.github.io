    # 数据结构与算法

    # 基本概念

    数据结构是相互之间存在一种或多种特定关系的数据元素的集合
    数据结构包括三方面: 逻辑结构, 存储结构和数据的运算.

    • 数据的逻辑结构
      集合: 类似于数学上的集合
      线性结构: 结构中的数据元素之间只存在一对一的关系
      树形结构: 结构中的数据元素之间存在一对多关系
      图状结构或网状结构: 结构中的数据元素之间存在多对多的关系
    • 存储结构(物理结构)
      顺序存储: 存储的物理位置相邻
      链式存储: 存储的物理位置未必相邻, 通过记录相邻元素的物理位置来找到相邻元素
      索引存储: 类似于目录
      散列存储: 通过关键字直接计算出元素的物理地址

    # 复杂度分析

    空间复杂度: 用来衡量算法随着问题规模增大, 算法所需空间增长的快慢
    时间复杂度: 用来衡量算法随着问题规模增大, 算法执行时间增长的快慢
    常见的时间复杂度大小关系:
    O(1) < O(logn) < O(n) < O(nlogn) < O(n^2) < O(n^3) < O(2^n)

    最坏

    # 基本算法思想

    算法的五大特征

    1. 有穷性: 有限步骤之后结束, 不会出现死循环
    2. 确定性: 不存在二义性, 算法的每个步骤被精确定义
    3. 可行性: 有些算法虽然理论可行, 但实际上无法完成
    4. 输入: 能被计算机处理的各种类型数据, 如数字, 音频, 图像等
    5. 输出: 一至多个程序输出结果

    # 选择排序(n^2)

    设置最小值的索引为0, 如果取出的值比当前最小值小, 则替换最小值索引, 遍历数组完成后, 将第一个元素和最小值索引上的值交换.

    function selectSort(arr) {
	let len = arr.length, minIdx = 0;
  for(let i = 0; i < len; i++){
  	for(let j = i+1; j < len; j++){
    	minIdx = arr[minIdx] > arr[j]? j : minIdx;
    }
    [arr[i], arr[minIdx]] = [arr[minIdx], arr[i]];
  }
}

    let arr = [6, 1, 2, 7, 9, 3, 4, 5, 10, 8];

function quickSort(arr, left, right) {
  if(left > right) return;
  let [base, i, j] = [left, left, right];
  while(i !== j) {
    while(arr[j] >= arr[base] && j > i)j--;
    while(arr[i] <= arr[base] && j > i)i++;
    [arr[i], arr[j]] = [arr[j], arr[i]];
  }
  [arr[base], arr[i]] = [arr[i], arr[base]];
  quickSort(arr, base, i-1);
  quickSort(arr, j+1, right);
}

    # 树

    位于树顶部的节点叫作根节点. 没有子元素的节点叫作叶子节点.
    树的高度取决于所有节点深度的最大值

    # 二叉树

    二叉树中的节点最多只能有两个子节点.
    二叉查找树是二叉树的一种, 它只允许你在左侧节点存储比父节点小的值, 在右侧节点存储比父节点大或者等于的值.

    class BinarySearchTree {
  constructor(key) {
    this.root = key || null;
  }

// 插入新节点
  insert = (value)=>{
    let newNode = new Node(value);
    if(this.root === null) this.root = newNode;
    else this.insertNode(this.root, newNode);
  }
  insertNode = (node, newNode)=>{
    if(newNode.value < node.value) {
      if(node.left === null) {
        node.left = newNode;
        newNode.father = node;
      }else this.insertNode(node.left, newNode);
    }
    else {
      if(node.right === null) {
        node.right = newNode;
        newNode.father = node;
      }else this.insertNode(node.right, newNode);
    }
  }

  // 求最小的节点
  min = ()=>{
  	return this.minNode(this.root);
  }
  minNode = (node)=>{
    if(node === null || node.left === null)return node.value;
    else return this.minNode(node.left);
  }
}

// 树的节点
class Node {
    constructor(value) {
      this.value = value;
      this.father = null; // 父节点
      this.left = null; // 左子节点
      this.right = null; // 右子节点
    }
}
