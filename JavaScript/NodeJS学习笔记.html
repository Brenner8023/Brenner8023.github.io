    # 简介

    官网: https://nodejs.org/zh-cn/

    1. 没有BOM, DOM
    2. ECMAScript
    3. 新增了一些服务器级别的操作API: 文件读写, 网络服务, 网络通信

    Node.js特性
    event-driven事件驱动
    non-blocking I/O model非阻塞IO模型(异步)

    # commonjs规范

    node应用由模块组成, 采用CommonJS模块规范.
    每个单独文件就是一个模块, 有自己的作用域, 对其它文件不可见. 在node中没有全局作用域.

    let a = '321';
let f = ()=>{};

// var exports = module.exports

exports.a =a; // 导出a
exports.f = f;

const bb = require('bb'); // 加载模块
bb.a
bb.f()

// 第一个参数是文件路径, 第二个参数是文件内容, 第三个参数是一个回调函数

    # HTTP服务

    const http = require('http');

// 返回一个server实例
const server = http.createServer();

// 注册request请求事件
server.on('request', (req, res)=>{});

// 绑定端口号, 启动服务
server.listen(3000, ()=>{});

// request请求对象
server.on('request', (req, res)=>{
	console.log('请求路径:'+req.url);
});

// response响应对象
server.on('request', (req, res)=>{
  res.write("hello");
	res.write("响应给客户端");
  res.end(); // 必须告诉客户端响应结束
});

// 简写模式
server.on('request', (req, res)=>{
  res.setHeader('Content-Type', 'text/plain; charset=utf-8');
  res.end(JSON.stringfy(arr));
});
