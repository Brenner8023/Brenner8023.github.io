                # 位图和矢量图

                位图: 位图是由像素点组合而成的图像, 一个点就是一个像素, 每个点都有自己的颜色. 分辨率大的位图清晰度高, 其放大倍数也相应增加. 但是当位图的放大倍数超过其最佳分辨率时, 就会出现细节丢失, 并产生锯齿状边缘的情况.

                矢量图: 矢量图是以数学向量方式记录图像的, 其内容以线条和色块为主. 矢量图和分辨率无关, 可以任意地放大且清晰度不变.

                svg是基于XML的矢量图形描述语言, svg是基于矢量图的, 而canvas是基于位图的

                # transform

                这个属性允许你将元素旋转, 缩放, 位移, 倾斜

                div {
	transform: rotate(10deg); // 旋转
  
  transform: skew(10deg, 11deg); // 扭曲
  
  transform: scale(2.0); // 缩放
  transform: scale(-1, -1);
  
  transform: translate(120px, 100px); // 向右平移120px, 向上平移100px
  transform: translate(50%, 50%); // 百分比以自身宽高为基准, 可实现未知宽高元素的水平垂直居中
  
  transform: matrix(a,b,c,d,e,f); // 矩阵
}

                # transition

                使css的属性值在一定的时间内平滑地过渡.

                div {
  background-color: red;
	transition: background-color 2s ease-in 1s;
}
div:hover {background-color: blue;}

transition-property: background-color; // 设置哪些属性进行过渡, 默认是all
transition-duration: 2s; // 完成过渡动画效果的时间, 默认是0
transition-delay: 1s; // 设置动画开始过渡的延迟时间, 默认是0
transition-timing-function: ease-in; // 设置动画的缓动效果, 默认是ease(逐渐变慢)
ease-in加速, ease-out减速, ease-in-out加速然后减速

                # animation

                animation-name: @keyframes动画的名称
animation-duration: 动画完成一个周期所花费的时间, 默认是0
animation-delay: 动画开始的延迟时间
animation-timing-function: 动画的速度曲线
animation-iteration-count: infinite; 动画被播放的次数, 默认是1
animation-direction: 动画是否在下一周期逆向地播放, 默认是normal
animation-play-state: 动画是否正在运行或暂停, 默认是running
aniamtion-fill-mode: 动画结束之后保持该状态不变

@keyframes changeColor {
  0% {background-color: red;}
  50% {background-color: green;}
  100% {background-color: blue;}
}

div:hover {
  animation-name: changeColor;
  animation-duration: 3s;
  animation-timing-function: ease-out;
  animation-iteration-count: infinite;
}

                # requestAnimationFrame

                <div class="box"></div>

<style>
.box {
    width: 100px; height: 100px;
    background-color: red;
}
</style>

<script>
const box = document.getElementsByClassName('box')[0];
let value = 0;
function move() {
    value += 5;
    box.style.transform = ['translateY(', value, 'px)'].join('');
    requestAnimationFrame(move);
}
requestAnimationFrame(move);
</script>
