          # flex

          采用flex布局的元素, 称为flex容器, 其所有子元素自动成为flex项目

          # 容器的属性

          flex-direction: 决定主轴的方向, 取值有row, row-reverse, column, column-reverse

          flex-wrap: 如果一条轴排不下, 定义如何换行, 取值有wrap, no-wrap, wrap-reverse

          flex-flow: row wrap; 简写形式

          justify-content: 定义了子项在主轴上的对齐方式, 取值有space-between, space-around, space-evenly, center, flex-start, flex-end

          align-items: 定义了子项在交叉轴上的对齐方式

          # 子项的属性

          order: 定义项目的排列顺序权重, 数值越小, 排列越靠前

          flex-grow: 2; 所占据的宽度

          # 自适应布局

          # 两列自适应

          <style>
  /** flex **/
  .father {
  	display: flex;
  }
  .son1 {
  	width: 100px;
    height: 100px;
  }
  .son2 {
  	flex: 1;
    height: 100px;
  }
  
  /** margin+float **/
  .son1 {
  	float: left;
    width: 100px;
    height: 100px;
  }
  .son2 {
  	margin-left: 100px;
    height: 100px;
  }
  
  /** float+BFC **/
  .son1 {
  	float: left;
    width: 100px;
    height: 100px;
  }
  .son2 {
  	overflow: hidden;
    height: 100px;
  }
</style>
<div class="father">
  <div class="son1"></div>
  <div class="son2"></div>
<div>

          # 三栏中间自适应

          <div class="left">左侧</div>
<div class="right">右侧</div>
<div class="middle">中间</div>

.left {
    width: 100px;
    float: left;
    background-color: red;
}
.right {
	width: 100px;
    float: right;
    background-color: blue;
}
.middle {
	overflow: hidden;
    margin-left: 100px;
    margin-right: 100px;
    background-color: green;
}

          # 布局居中

          非块状元素

          .son {
    height: 50px;
    line-height: 50px;
    text-align: center;
}

          垂直水平居中

          .father {
    display: flex;
    justify-content: center;
    align-items: center;
}

.father {
    display: flex;
}
.son {
    margin: auto;
}

          未知元素宽高的居中

          .father {
    position: relative;
}
.son {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
}
