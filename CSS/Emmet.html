    官网: https://emmet.io/

    emmet使用类似css选择器的语法来描述元素在DOM树节点的位置和属性, 通过使用这样一种缩写语法来快速编写html, css, 提高前端的开发效率.

    例如:
div#page>ul.nav$@3*2>li*4>a{啦啦啦 $}

会被解析成:
<div id="page">
  <ul class="nav3">
    <li><a href="">啦啦啦 1</a></li>
    <li><a href="">啦啦啦 2</a></li>
    <li><a href="">啦啦啦 3</a></li>
    <li><a href="">啦啦啦 4</a></li>
  </ul>
  <ul class="nav4">
    <li><a href="">啦啦啦 1</a></li>
    <li><a href="">啦啦啦 2</a></li>
    <li><a href="">啦啦啦 3</a></li>
    <li><a href="">啦啦啦 4</a></li>
  </ul>
</div>
    • 创建模板和导入
    html:5
会被解析成:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
</body>
</html>

link:css
会被解析成:
<link rel="stylesheet" href="style.css">

script:src
会被解析成:
<script src=""></script>
    • 生成元素子节点
    div>ul>li

会被解析成:
<div>
  <ul>
    <li></li>
  </ul>
</div>
    • 生成相邻兄弟结点
    div+p+a

会被解析成:
<div></div>
<p></p>
<a></a>
    • 生成多个相同的元素
    ul>li*3

会被解析成:
<ul>
	<li></li>
  <li></li>
  <li></li>
</ul>
    • 使用圆括号
    (div>dl>(dt+dd)*3)+footer>p

会被解析成:
<div>
  <dl>
    <dt></dt>
    <dd></dd>
    <dt></dt>
    <dd></dd>
    <dt></dt>
    <dd></dd>
  </dl>
</div>
<footer>
  <p></p>
</footer>
    • 给元素添加id和class
    div#header.class1.class2.class3

会被解析成:
<div id="header" class="class1 class2 class3">
</div>
    • 添加其它属性键值对
    td[title="hello world" colspan=3]

会被解析成:
<td title="hello world" colspan=3></td>
    • 给元素进行有序编号
    ul>li.item$*3
会被解析成:
<ul>
  <li class="item1"></li>
  <li class="item2"></li>
  <li class="item3"></li>
</ul>

ul>li.item$$$*3
会被解析成:
<ul>
  <li class="item001"></li>
  <li class="item002"></li>
  <li class="item003"></li>
</ul>

ul>li.item$@3*3
会被解析成:
<ul>
  <li class="item3"></li>
  <li class="item4"></li>
  <li class="item5"></li>
</ul>
    • 给元素添加文本
    a{点击我}

会被解析成:
<a>点击我</a>
