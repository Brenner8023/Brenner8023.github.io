            # SCSS

            css预处理器定义了一种新的语言, 其基本思想是, 用一种专门的编程语言, 为css增加一些特性, 将css作为目标生成文件.

            sass和scss其实是同一种东西, 两者之间的不同之处有:

            1. 文件扩展名不同, sass是以.sass后缀为扩展名, 而scss是以.scss后缀为扩展名
            2. sass是以严格的缩进式语法规则来书写, 而scss的语法书写和css语法书写比较类似
            npm i -S node-sass

            # 混合宏

            类似于函数

            // 声明混合宏
@mixin border-raduis($radius: 5px) {
	-webkit-border-radius: $radius;
  border-radius: $radius;
}

// 调用混合宏
button {
	@include border-radius;
  // @include border-radius(50%);
}

// 传多个参数
@mixin center($width, $height) {
	width: $width;
  height: $height;
}

            # 继承

            .btn {
  border: 1px solid #ccc;
  padding: 6px 10px;
  font-size: 14px;
}

.btn-primary {
  background-color: #f36;
  color: #fff;
  @extend .btn;
}

// 占位符
%mt5 {
	margin-top: 5px;
}
.btn {
	@extend %mt5;
}
